Um laboratório de estratégias de negociação algorítmica (trading automatizado) desenvolvido em **MQL5**, com foco em análise técnica, indicadores e Machine Learning.

## 📌 Funcionalidades

- ✅ Estratégias baseadas em indicadores técnicos
- ✅ Backtesting integrado com MetaTrader 5
- ✅ Gerenciamento de risco
- ✅ Modular e expansível
- ✅ Logs e monitoramento das operações

## Tecnologias

- **MQL5** (MetaQuotes Language 5)
- MetaTrader 5
- Git + GitHub

## Como usar

1. Clone este repositório:

```bash
git clone https://github.com/Gigio42/AlgoTradingLab.git
```
2. Copie o projeto para a pasta de Experts do MetaTrader 5:
```bash
C:\Users\SeuNome\AppData\Roaming\MetaQuotes\Terminal\...\MQL5\Experts\
```
3. Compile os arquivos .mq5 no MetaEditor.

4. Faça o backtest ou execute no gráfico do ativo desejado.
